# Image Blur Tool

웹 브라우저에서 실행되는 이미지 블러 처리 프로그램입니다.

## 기능

- 이미지 불러오기 (.gif, .jpg, .jpeg, .png, .tif, .tiff)
- 마우스 드래그로 블러 영역 선택
- 가변 블러 강도 설정 (1~50)
- 블러 적용 되돌리기 기능
- 처리된 이미지 저장

## 사용법

1. `imageBlur.html` 파일을 웹 브라우저에서 열기
2. "불러오기" 버튼으로 이미지 선택
3. 블러 강도 설정
4. 마우스로 블러를 적용할 영역 드래그
5. "저장" 버튼으로 처리된 이미지 다운로드

## 기술 스택

- HTML5
- CSS3
- JavaScript (Vue.js 3)
- Canvas API
